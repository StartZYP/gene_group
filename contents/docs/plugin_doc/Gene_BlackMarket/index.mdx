## 功能
1.回收

支持多货币

支持多配置

支持回收多个

2.轮换黑市(出售)

支持多货币

限购(有的物品限购有的物品不限购)

时间周期轮换 (秒) 全服统一 商品限购也刷掉

物品池 (可以配置每个物品概率)

支持多个



3.非轮换黑市(商店)
支持多货币
无限购
支持多个
