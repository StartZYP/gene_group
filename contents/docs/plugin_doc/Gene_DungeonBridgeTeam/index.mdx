## 发包文档

<Step>
<StepItem title="数据首包">

首包: Gene_DungeonBridgeTeamView

界面文件名: dungeonBridgeViewScreen

</StepItem>

<StepItem title="副本展示数据">

数据包1: dungeon_key

数据包2: 副本关键key

---

变量名: GeneDungeonBridgeTeamView_DungeonInfo

返回副本数据:

GeneDungeonBridgeTeamView_DungeonKey (副本名)

GeneDungeonBridgeTeamView_自定义数据key (配置中自定义)

GeneDungeonBridgeTeamView_1_Time (队长通关排名数据)

GeneDungeonBridgeTeamView_2_Time

GeneDungeonBridgeTeamView_3_Time

GeneDungeonBridgeTeamView_Me (自己的通关数据)

槽位变量名: GeneDungeonBridgeTeamView_Slot_索引

</StepItem>

<StepItem title="进副本">

数据包1: enter

数据包2: 副本关键key

数据包3: 副本困难程度  (1,2,3) 数字分别对应简单 中等 困难

---

无返回数据,通过title提示队员是否满足进服数据

</StepItem>

</Step>


# 配置Kether变量

CareerLevel (生涯等级)

CombatPower (战斗力)

Max (最大人数限制)

Min (最小人数限制)

TeamNumber (队伍人数)

# Kether脚本注意事项

切记返回要是字符串

如果判断全部成功则返回 true

判断失败返回 false

判断中出问题则返回 一句话用于提示全队

# 配置中副本难度

PEACEFUL(0),

EASY(1),

NORMAL(2),

HARD(3);

切记都是字符串
