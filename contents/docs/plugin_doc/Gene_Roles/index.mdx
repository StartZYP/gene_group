## 发包文档

<Step>
<StepItem title="命令打开">

命令打开: grs open [玩家]

界面名称: Gene_Roles

handle处理名称: data

收包:

Gene_CurrentRole 当前在用的角色

Gene_Role_Size 拥有角色总数量

Gene_Role_[index] 拥有角色名称

Gene_Role_[index]_AttSize 角色属性词条数量

Gene_Role_[index]_Att_[index] 第一个角色索引 第二个角色属性条数索引
