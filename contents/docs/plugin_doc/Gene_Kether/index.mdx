## Kether 函数文档

### SendImageTitle
```
SendImageTitle (player) (path) (width) (height) (keepms) (soundpath)
```
- **参数说明:**
  - `player`: 玩家名称
  - `path`: 图片路径
  - `width`: 图片宽度
  - `height`: 图片高度
  - `keepms`: 显示持续时间(毫秒)
  - `soundpath`: 音效路径

### SendMessage
```
SendMessage (player) (message)
```
- **参数说明:**
  - `player`: 玩家名称
  - `message`: 要发送的消息内容

### SendCaption
```
SendCaption (player) (headPath) (name) (message) (keepMS) (soundpath)
```
- **参数说明:**
  - `player`: 玩家名称
  - `headPath`: 头像路径
  - `name`: 显示名称
  - `message`: 消息内容
  - `keepMS`: 显示持续时间(毫秒)
  - `soundpath`: 音效路径
